# Catálogo Digital 4C – Primera Promoción

Este proyecto es la transposición digital de la portada del catálogo físico de la **1ª Promoción de 4C**. El objetivo principal ha sido mantener la integridad del diseño editorial original, trasladando la precisión tipográfica y compositiva de InDesign al entorno web (HTML5/CSS3).

## El Desafío de Diseño
Como diseñadora, el reto fue replicar el sistema de cuadrículas y jerarquías tipográficas en un formato dinámico. 
* **Fidelidad Tipográfica:** Implementación de fuentes específicas mediante `@font-face` (**Coolvetica** y **Nyght Serif**) para respetar la identidad visual de la marca.
* **Composición Áurea:** Uso de posicionamiento absoluto y Flexbox para clavar las coordenadas de los nombres de los artistas, el logo y el escalonado diagonal del título.
* **Responsive Design:** Adaptación de la pieza a diferentes resoluciones para que la experiencia sea fluida tanto en desktop como en dispositivos móviles.

## Tecnologías utilizadas
* **HTML5:** Estructura semántica.
* **CSS3:** Maquetación avanzada, custom properties y media queries.
* **Tipografías:** Coolvetica (Regular/Italic) y Nyght Serif (Light Italic).

## Instalación y Vista Previa
Para ver el proyecto en local:
1. Clona el repositorio.
2. Abre el archivo `index.html` en tu navegador.
